# Changelog

All notable changes to the Royal Fit Uniform website are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-06-30

### Added
- Multi-page marketing website with 6 pages: Home, Solutions, Lookbook, Our Story, Contact, Product
- Material Design 3 token-based design system with Tailwind CSS 4
- Smooth client-side page transitions via Swup with fade theme
- Product catalog with 270+ optimized WebP images
- SEO setup: unique meta tags, Open Graph, JSON-LD structured data, sitemap.xml, robots.txt
- Responsive layout for mobile, tablet, and desktop
- Vercel deployment with clean URLs, security headers, and asset caching
- PWA support with web manifest and app icons
- Catalog image processing script (`process_catalog.cjs`)
- Lookbook data update script (`update_lookbook.js`)
